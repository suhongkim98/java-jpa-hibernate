# java-jpa-hibernate
`JPA Hibernate`의 다양한 사용 예시를 담은 레포지토리입니다.
```
.
├── main
│   ├── java
│   │   └── domain
│   │       ├── Person.java
│   │       ├── School.java
│   │       ├── SchoolClass.java
│   │       └── type
│   │           └── SchoolRank.java
│   └── resources
│       └── META-INF
│           └── persistence.xml
└── test
    └── java
        ├── CascadeTests.java
        ├── FetchJoinTests.java
        ├── HibernateTests.java
        └── JoinTests.java
```
